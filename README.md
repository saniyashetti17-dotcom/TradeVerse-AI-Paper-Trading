# 🚀 TradeVerse - AI Powered Paper Trading Platform

## 📌 Overview

TradeVerse is an AI-powered paper trading platform that allows users to practice stock market investing with virtual money while receiving AI-generated trading signals and portfolio recommendations.

The platform provides live stock prices, AI-based buy/sell recommendations, portfolio analytics, watchlists, commodity tracking, and market insights.


### Frontend

```bash
npm install
npm run dev
```
